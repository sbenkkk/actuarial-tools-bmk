# Contrato de datos — Actuarial Tools by BMK

Nombres de columna estándar compartidos entre herramientas (apartado 7.6).
Permiten que el output de una herramienta alimente a otra dentro de Internal
Model Studio. **Nunca se improvisa un nombre a nivel de herramienta**: todo
nombre estándar nuevo se añade aquí primero.

| Concepto | Nombre de columna |
|----------|-------------------|
| Importe de siniestro / severidad | `loss_amount` |
| Fecha de ocurrencia | `date_occurred` |
| Identificador de póliza | `policy_id` |
| Identificador de siniestro | `claim_id` |
| Exposición | `exposure` |

Lista ampliable. Cada ampliación se documenta en esta tabla.
